Changements dans la version 3.3.0

  • Version Portugaise/Brésilienne grâce à Manuel Fernandes
    Para os usuarios da lingua Portuguesa: devera baixar o DSS integralmente em portugues, juntamente com o ficheiro de ajuda, tambem em portugues. .
     
  • Amélioration du déblooming des grandes étoiles bloomées
  • Amélioration du débayering CYMG
  • Ajout de 16 motifs CYMG génériques (onglet FITS DDP)
  • Ajout du support des fichiers TIFF contenant 4 canaux (le canal contenant la transparence est ignoré).
  • Ajout d'une option d'empilement Sans Alignement  (les images sont empilées telle quelle sans rotation/translation)
  • Ajout d'un mode intersection pour créer une image finale qui est l'intersection de toutes les brutes (et supprimer les bandes noires)
  • Ajout d'un mode 4 coins pour aisément vérifier les 4 coins d'une image (disponible également dans DeepSkyStacker Live)
  • Le format de l'image finale est maintenant le même que celi utilisé pour les images intermédiaires (FITS ou TIFF)
  • L'extension FITS utilisée pour les fichiers intermédiaires et la même que celle des images d'origine lorsque qu'un format FITS est sélectionné (.fts, .fit ou .fits)

Corrections :
 Le fichier de description html n'était pas créé lors de l'utilisation de l'empilement par lot. Corrigé.
Les fichiers NEF des Nikon Coolpix 5000 étaient incorrectement décodés comme utilisant une matrice de bayer RGB au lieu de CYMG (d'autres APN pouvaient être dans le même cas). Corrigé.
Le mode de calbration du fond n'était pas écrit dans le fichier de description html. Corrigé.
DSS Live: la profondeur était incorrectement montrée comme 16 bit indépendament de la vraie profondeur (toutefois cela ne changeait pas le traitement). Corrigé.
DSS Live: Le lien pour copier l'image courante n'était pas disponible dans l'onglet Image empilée. Corrigé.


Changements des versions précédentes

Installation : décompactez DeepSkyStacker.zip dans n'importe quel répertoire et lancer Setup.exe.
Si vous rencontrez un problème lors de l'installation (dll absente ...), assurez vous que vous avez la dernière version de Windows Installer en le téléchargeant ici. Ceci peut particulièrement être le cas avec Windows XP sans le SP2.

Plates-formes supportées : Windows XP SP2, Windows 2000, Windows 2003 Server, Windows Vista
(cela fonctionne peut être avec Windows 98 et Windows NT 4 mais je n'ai jamais eu l'occasion d'essayer).

La présence de GDI+ est indispensable. Si vous ne possédez pas Windows XP, Windows 2003 Server ou Windows Vista, cliquez ici pour télécharger GDI+ pour votre plate-forme.

Résolution minimale : 1024x768

Mémoire nécessaire : en fonction de la taille des images.
256 Mo si les images sont relativement petites (moins de 1000x1000 pixels)
512 Mo pour des images venant d'un appareil photo numérique type Canon EOS 350D
Plus ne peut pas faire de mal (cela évite d'utiliser le swap et accélère les traitements)

Processeur : à partir d'un PIII 800 Mhz. Comme toujours le plus puissant est le mieux si vous souhaitez accélérer les traitements.
