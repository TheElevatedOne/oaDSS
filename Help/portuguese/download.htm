  • Adicionou-se 16 genericos CYMG modelos bayer (FITS DDP tab)
  • Adicionou-se suporte para 4 canais em ficheiros TIFF (o canal transparencia e ignorado).

  • O final do formato do ficheiro da imagem de producao, e agora o mesmo do que foi usado para os ficheiros intermedios (seja TIFF ou FITS)
  • Extensao do ficheiro FITS abrange ate aos ficheiros intermedios e/ou quando a imagem final e nomeada FITs como ficheiro de producao selecionado (.fts, .fit or .fits)

Problemas corrigidos:
 Discricao do ficheiro HTML nao era criado quando se fazia a integracao em série.Resolvido.
Nikon Coolpix 5000- Ficheiros NEF eram descodificados incorrectamente usando o modelo RGB da Bayer em vez do modelo CYMG (outras DSLRs usam CYMG modelo da Bayer...podera ter sido alguns dos casos). Resolvido.
O modo de calibracao da tela do fundo, nao foi escrita na discricao HTML do ficheiro. Resolvido.
DSS Live: Mostrava incorrectamente 16 bits na lista de ficheiros omitindo a verdadeira profundidade do ficheiro (isto nao interferia com o processo). Resolvido.
DSS Live: Ligacao de copia para o clipboard nao estava disponivel na Aba Integracao de Imagem. Resolvido.

Previos apontamentos

Instalacao: Apenas descompacte o DeepSkyStacker.zip numa pasta e acione o Setup.exe.
Em caso de problema na instalacao(algum ficheiro dll que falte...) verifique se esta usando a ultima do windows here. ou o seu Windows XP nao tem o SP2.

Protocolo de Plataformas: Windows XP SP2, Windows 2000, Windows 2003 Server, Windows Vista
(podera funcionar com Windows 98 e Windows NT 4 mas nunca foi testado nestas plataformas por isso nao o passo garantir).
GDI+ tem que ser instalado. Se nao esta a usar o Windows XP, Windows 2003 Server ou Windows Vista, clique aqui para baixar o GDI+ para sua plataforma
(GDI+ esta instalado com Windows XP, Vista e 2003 Server)

Minimo de resolucao: 1024x768

Memoria requerida: dependendo do tamanho das fotos .
256 Mb para pequenas fotografia (menos de 1000x1000 pixels)
512 Mb para fotografias feitas com DSLRs como a Canon EOS 350D
... mas claro, quanto mais memoria tiver no PC melhor(melhora a velocidade de processamento)

Processador: acima de 800 MHz PIII.
De novo, quanto mais rapido for o processador, mais rapido acaba o pre-processamento.
