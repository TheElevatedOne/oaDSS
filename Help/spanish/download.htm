Cambios en la versión 3.3.0

  • Version en Portugues/Brasilero gracias a Manuel Fernandes
    Para os usuarios da lingua Portuguesa: devera baixar o DSS integralmente em portugues, juntamente com o ficheiro de ajuda, tambem em portugues. .
     
  • Deblooming mejorado para estrellas con bloom grande
  • Debayerizado CYMG mejorado
  • Agregado 16 patrones genéricos de bayer CYMG (pestaña FITS DDP)
  • Agregado soporte para archivos TIFF de 4 canales (el canal de transparencia es ignorado)
  • Agregada la opción de No Alineado (las imágenes son apiladas tal cual estan, sin rotación/translación)
  • Agregado el Modo Intersección para crear una imagen final la cual es la intersección de todos los cuadros (por lo tanto removiendo los bordes negros)
  • Agregado el Modo 4 esquinas para chequear fácilmente las esquinas de una imagen (también disponible en DeepSkyStacker Live)
  • El formato del archivo de salida final ahora es el mismo que se utiliza para los archivos intermedios (tanto TIFF como FITS)
  • La extensión de archivo FITS es propagada a los archivos intermedios y al nombre de la imagen final cuando se selecciona salida en formato FITS (.fts,  .fit o .fits)

DSS Live.: La profundidad de bits se mostraba en la lista de archivos de manera incorrecta como 16 bits independientemente de la verdadera profundidad de bits (De todos modos no modificaba el proceso). Corregido.

Notas de versiones anteriores

Instalación: solo descomprima DeepSkyStacker.zip en cualquier carpeta y ejecute Setup.exe.
En caso de ocurrir algún inconveniente (pérdida de alguna librería dll...) verifique que está utilizando el último Instalador de Windows que puede descargarse aquí. Esto puede suceder en el caso especial que usted esté utilizando Windows XP sin SP2.

Plataformas Soportadas: Windows XP SP2, Windows 2000, Windows 2003 Server, Windows Vista
(puede funcionar con Windows 98 y Windows NT 4 pero nunca han sido probadas y no puedo garantizar su funcionamiento).
GDI+ debe estar instalado. Si usted no está utilizando Windows XP, Windows 2003 Server o Windows Vista, haga click aquí para descargar GDI+ para su plataforma
(GDI+ está instalado con Windows XP, Vista y 2003 Server)

Resolución mínima: 1024x768

Requirimiento de memoria: depende del tamaño de las imágenes.
256 Mb para imágenes pequeñas (menos de 1000x1000 píxeles)
512 Mb para imágenes creadas con cámaras DSLRs similares a Canon EOS 350D
Por supuesto que, cuanto más memoria tenga, es mejor (Se evita el uso de swap y acelera los procesos)

Procesador: desde un PIII de 800 MHz.
Nuevamente, si usted desea acelerar los procesos, cuanto más rápido trabaje el microprocesador, mejor es.
